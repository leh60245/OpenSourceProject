## Docs + chat
Docs + chat은 기존 google docs가 제공하는 서비스에 채팅 기능까지 더해진 웹사이트입니다. 실시간으로 팀 구성원들과 의사소통하며 문서를 작성하고 수정해보세요!

###주요기능
- 회원가입, 로그인
- 공동 문서 작업(Google Docs)
- 사용자간 채팅

## Docs + chat 설계
### 회원가입 / 로그인
####Requirements
-XAMPP for 7.4.7

  -MariaDB 10.4.13
  -PHP 7.4.7
